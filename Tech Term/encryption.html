Encryption is the process of taking the message and scrambling it so that only certain people can look at the message.

Symmetric Encryption

  • Symmetric Encryption is a type of encryption where only one key (a secret key) is used to both encrypt and decrypt data.
  • The entities communicating via symmetric encryption must exchange the key so that it can be used in the decryption process. 
  • AES, DES, IDEA, Blowfish, RC5 and RC6  and RC4 are cipher used in Symmetric encryption.
  •  Encrypted message are called ciphertext

Asymmetric Encryption

  • Suppose two person A and B wants to transfer message.
  • First both A and B generates its own Private and Public Key. Lets say key are created using RSA algorithm.
  • Note:Private key and Public key are mathematically linked to each other. But they cannot be derived from each other. ie. If you know someone public key you cannot derived private key. Also Note, message can be locked and unlocked with same public/private key.
  • Public Key is always used to encrypted the data and matching private key can only decrypted the data.
  • Now public key of A and B are broadcast. So that A received Public Key of B and B received public Key of A.
  • Now suppose A wants to send message to B, A encrypt his message with B public key and send encrypted message to B . Now B recieved the encrypted message which B decrypt using her Private Key.Not even A can decrypt this message.
  • So main things is A and B has to keep thsse private and public key secretly.
  • Used in Https web,SSL, SSH, Bitcoin, PGP
